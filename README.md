# GazeGuard-

#A study companion that measures attention in real time
Use webcam and computer vision to track movement of head and eyes
User can set an amount of time to study 
When eyes are not directly focused on the screen, a warning display takes over the screen to alert the user to focus back on work
Keep track of how many times the focus alert is triggered in a set study block and record it
As the user keeps using the app to study, insights can be provided based on their trends
Ex. avg. amount of times distracted, has their attention span gotten better
